
#### 介绍
-----config 项目配置
|    |-----dev.js  开发时配置
|    |----index.js 默认配置
|    |----prod.js  打包时配置
|----src 源码目录
|    |----assets 静态文件
|    |     |----iconfont 图标
|    |     |----images   图片
|    |     |----styles   公共样式
|    |-----components 公共组件
|    |-----pages 页面文件目录
|    |     |-----index index页面目录
|    |           |----component 私有组件
|    |           |----index.tsx 页面逻辑
|    |           |----index.scss 页面样式     
|    |-----utils 公共方法库      
|    |-----app.jsx 项目入口文件      
|    |-----app.scss 项目总通用样式      
-----.eslintrc eslint 配置
-----package.json 依赖包
-----project.config.json 工具配置
-----tsconfig.json 配置
